# Kraken

> Kraken is one of the world's largest and longest-running cryptocurrency exchanges (founded 2011, San Francisco). It exposes a comprehensive API surface — REST, WebSocket, and FIX — across the Spot exchange, Futures exchange (Kraken Futures, formerly Crypto Facilities), institutional services (Custody, OTC, Prime), and partner integration surfaces (Embed, Ramp, OAuth). Spot and Futures private endpoints authenticate with API-key HMAC-SHA512 signatures and an incrementing nonce; errors are returned in a {error, result} envelope, not HTTP status codes.

This file was generated by the API Evangelist enrichment pipeline from apis.yml and the artifacts in this repository. Kraken does not publish an official llms.txt; verify anything auth- or funds-related against the live docs at https://docs.kraken.com/api/.

## APIs
- [Kraken Spot REST API](https://docs.kraken.com/api/docs/rest-api/get-server-time): Public market data + private account/trading/funding/Earn on api.kraken.com.
- [Kraken Spot WebSocket API v2](https://docs.kraken.com/api/docs/category/websocket-v2/): Real-time market and private data with authenticated trading on wss://ws.kraken.com/v2.
- [Kraken Spot WebSocket API v1 (Legacy)](https://docs.kraken.com/api/docs/category/websocket-v1): Legacy WebSocket; v2 recommended for new work.
- [Kraken Futures REST API](https://docs.kraken.com/api/docs/futures-api/trading/get-accounts): Derivatives trading, market data, and account data on futures.kraken.com/derivatives/api/v3.
- [Kraken Futures WebSocket API](https://docs.kraken.com/api/docs/category/futures-websocket): Real-time futures feeds.
- [Kraken FIX API](https://docs.kraken.com/api/docs/category/fix-api/): FIX 4.4 / 5.0 SP2 for institutional execution.
- [Kraken NFT API](https://docs.kraken.com/api/docs/category/nft-rest/): NFT marketplace surfaces.
- [Kraken Earn API](https://docs.kraken.com/api/docs/rest-api/allocate-strategy): Staking and yield allocation.
- [Kraken Embed REST API](https://docs.kraken.com/api/docs/category/embed/): B2B embedded trading/custody/Earn.
- [Kraken OAuth REST API](https://docs.kraken.com/api/docs/category/oauth/): OAuth 2.0 delegated access.
- [Kraken Ramp / Custody / OTC / Prime](https://docs.kraken.com/api/): Fiat on/off-ramp and institutional surfaces.
